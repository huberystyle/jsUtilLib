# jsUtilLib
这是一个常用的js工具函数库，一点一点追加！


使用说明：
```js
jsUtilLib._getWinSize();  // 获取窗口视口高度 和 宽度，以数组形式返回。
jsUtilLib._getDocSize(); // 获取页面高度 和 宽度，以数组形式返回。
jsUtilLib._getUrlparameter("name"); // 返回url中名字为name的参数值
jsUtilLib.tab("tabli","tabcontent"); // tab选项卡
jsUtilLib.fmoney("s","n"); // 将一段数字转换成 1000,555,22.00 格式，俩参数 s 字符串， n 四舍五入小数位
jsUtilLib.rmoney("s"); // 将人民币格式转换成数字，s 要转换的字符串
```
